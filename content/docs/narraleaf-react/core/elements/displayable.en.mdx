---
title: "Displayable"
---
Displayable defines all the properties and methods that a displayable element should have. Displayable elements are elements that can be displayed on the screen.  
Ex. Image, Text

## Chainable Methods

### pos

<Steps>
<Step>

Transform the element to a new position.

```typescript
element.pos({
    xalign: 0.3
}, 1000, "linear"); // Move the element to x: 0.3 in 1000ms with linear ease.
```

- `position: CommonDisplayableConfig["position"]` - The new position of the element, see [CommonDisplayableConfig](../types/CommonDisplayable#position)
- `duration?: number` - The duration of the transform in milliseconds
- `easing?: TransformDefinitions.EasingDefinition` - The easing function to use, see [TransformDefinitions](../types/TransformDefinitions).[EasingDefinition](../types/TransformDefinitions#EasingDefinition)
</Step>
</Steps>

### scale

<Steps>
<Step>

Scale the element.

```typescript
element.scale(1.5, 0.8, 1000, "easeInOut"); // Scale the element to 1.5x on X-axis and 0.8x on Y-axis in 1000ms with easeInOut ease.
```

- `scaleX: number` - The new scale of the element on the X-axis, use negative value to invert the scale
- `scaleY: number` - The new scale of the element on the Y-axis, use negative value to invert the scale
- `duration?: number` - The duration of the transform in milliseconds
- `easing?: TransformDefinitions.EasingDefinition` - The easing function to use, see [TransformDefinitions](../types/TransformDefinitions).[EasingDefinition](../types/TransformDefinitions#EasingDefinition)
</Step>
</Steps>

### show

<Steps>
<Step>

#### Overload 1 of 3

Show the element immediately.

```typescript
element.show();
```

#### Overload 2 of 3

Show the element with custom transform.

```typescript
element.show({
    /* Transform Definitions */
});
```

- `options: Transform` - The custom transform to apply to the element, see [Transform](../animation/transform)

#### Overload 3 of 3

Show the element with transform config. A transform with alpha:1 will be applied to the element.

```typescript
element.show({
    duration: 1000,
    ease: "easeInOut",
});
```

- `options: Partial<TransformDefinitions.CommonTransformProps>` - See [TransformDefinitions](../types/TransformDefinitions).[CommonTransformProps](../types/TransformDefinitions#CommonTransformProps)
</Step>
</Steps>
