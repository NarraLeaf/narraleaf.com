视频是一个元素，允许您播放视频。

视频是占用整个舞台的播放器。

```typescript
const video = new Video({
    src: "https://example.com/video.mp4",
    muted: true,
});

scene.action([
    video
        .show()
        .play()
        .hide(),
]);
```

## 链式方法

### show

<Steps>
<Step>

显示视频。

```typescript
video.show();
```
</Step>
</Steps>

### hide

<Steps>
<Step>

隐藏视频。
</Step>
</Steps>

### play

<Steps>
<Step>

播放视频并等待其完成。

```typescript
video.play();
```
</Step>
</Steps>

### pause

<Steps>
<Step>

暂停视频，并保留当前的播放位置。

```typescript
video.pause();
```
</Step>
</Steps>

### resume

<Steps>
<Step>

从当前位置继续播放。与 `play` 不同，它不会等待视频播放结束。

```typescript
video.resume();
```
</Step>
</Steps>

### stop

<Steps>
<Step>

停止视频：暂停播放，并结束任何正在等待的 `play()`，让剧情继续推进。

```typescript
video.stop();
```
</Step>
</Steps>

### seek

<Steps>
<Step>

- `time: number` - 要跳转到的时间（单位：秒）。

跳转到指定的时间点。

```typescript
video.seek(3);
```
</Step>
</Steps>
