视频是一个元素，允许您播放视频。

视频是占用整个舞台的播放器。

```typescript
const video = new Video({
    src: "https://example.com/video.mp4",
    muted: true,
});

scene.action([
    video
        .show()
        .play()
        .hide(),
]);
```
