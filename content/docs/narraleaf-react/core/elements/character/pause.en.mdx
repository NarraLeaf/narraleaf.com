---
title: "Pause"
---
Pause allows you to create pause in sentences.

```ts
new Character("character 1")
    .say([
        "So you mean...",
        Pause.wait(1000),
        "I can do this?"
    ])
```

or a pause that requires user to click to continue

```ts
new Character("character 1")
    .say([
        "I said: ",
        Pause,
        "I can do this!"
    ])
```

These two are equivalent: 

```ts
new Pause({duration: 1000});
// or
Pause.wait(1000);
```

And the following are equivalent:

```ts
new Pause();
// or
Pause;
```
