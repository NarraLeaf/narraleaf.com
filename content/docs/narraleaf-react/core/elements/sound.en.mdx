Sound controls the sound of the game

## Static Method

### voice

<Steps>
<Step>

Create a voice sound

```ts
const voice = Sound.voice({
    src: 'voice.mp3',
});

// is equivalent to
const voice = new Sound({
    src: 'voice.mp3',
    type: SoundType.Voice,
});
```

- `arg0: Partial<ISoundUserConfig> | string` - The config of the sound or the source of the sound, see [ISoundUserConfig](/docs/narraleaf-react/core/types/ISoundUserConfig)
- Returns `Sound`
</Step>
</Steps>

## Chainable Method

### play

<Steps>
<Step>

Play the sound and wait for it to finish

> **Note**: Sounds marked as `Bgm` cannot be played directly.

```ts
const sound = new Sound({
    src: 'sound.mp3',
});
scene.action([
    sound.play(1000), // play and fade in 1 second
]);
```

- `duration?: number` - Fade duration in milliseconds
</Step>
</Steps>

### stop

<Steps>
<Step>

Stop the sound. After stopping, the sound will go back to the beginning

- `duration?: number` - Fade duration in milliseconds
</Step>
</Steps>

### fade

<Steps>
<Step>

Fade the sound

- `start: number | undefined` - Start volume, from 0 to 1, default is current volume
- `end: number` - End volume, from 0 to 1
- `duration: number` - Duration in milliseconds

```ts
const sound = new Sound({
    src: 'sound.mp3',
    volume: 0,
});
scene.action([
    sound
        .play()
        .fade(0, 1, 1000), // fade in 1 second
]);
```
</Step>
</Steps>

### setVolume

<Steps>
<Step>

Set the volume of the sound

- `volume: number` - Volume, from 0 to 1
- `duration?: number` - Fade duration in milliseconds
</Step>
</Steps>

### mute

<Steps>
<Step>

Mute or unmute the sound without stopping playback.

```ts
scene.action([
    bgm.play(),
    bgm.mute(),
    bgm.unmute(),
]);
```

- `muted?: boolean` - `true` to mute the sound, `false` to restore volume (defaults to `true`)
- Returns `ChainedSound` - Chained Sound instance
</Step>
</Steps>

### unmute

<Steps>
<Step>

Alias of `mute(false)` to restore the original volume.

- Returns `ChainedSound` - Chained Sound instance
</Step>
</Steps>
