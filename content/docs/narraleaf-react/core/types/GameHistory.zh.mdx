---
title: "GameHistory"
---
```typescript
export type GameHistory = GameHistoryAction & {
    element: GameElementHistory;
    snapshot?: SerializedGameState | null;
};
```

`GameHistory` 类型表示一个完整的游戏历史条目，结合了动作信息和受影响的特定游戏元素。它扩展了 [GameHistoryAction](/zh/docs/narraleaf-react/core/types/GameHistoryAction) 并添加了：

- `element`: 作为此历史条目一部分的特定游戏元素（"say" 或 "menu" 元素），参见 [GameElementHistory](/zh/docs/narraleaf-react/core/types/GameElementHistory)
- `snapshot`: 该行处游戏状态的自包含快照，供 [restoreToHistory](/zh/docs/narraleaf-react/core/game/live-game#restoretohistory) 跳回此处使用。它随回溯增长自动填充并随存档持久化，无需自行构造。

更多信息，请参阅：
- [GameHistoryAction](/zh/docs/narraleaf-react/core/types/GameHistoryAction)
- [GameElementHistory](/zh/docs/narraleaf-react/core/types/GameElementHistory)
