---
title: "VfxConfig"
---
```typescript
type VfxConfig = {
    src: string;
    blendMode: VfxBlendMode;
    loop: boolean;
    muted: boolean;
    opacity: number;
    playbackRate: number;
    fit: "cover" | "contain" | "fill";
    zIndex: number;
};

type VfxBlendMode =
    | "normal"
    | "screen"
    | "multiply"
    | "lighten"
    | "color-dodge"
    | "overlay";
```

- `src` — 视频 URL。必填。
- `blendMode` — 叠加层与舞台的混合方式。默认 `"normal"`。`"normal"` 用于真 alpha 素材（VP9 `yuva420p` alpha WebM）；`"screen"` 是加法混合，用于绘制在黑底上的发光素材（暗色像素在亮背景上会被冲蚀）；`"multiply"` 用于绘制在白底上的阴影素材。
- `loop` — 是否循环播放。默认 `true`。
- `muted` — 默认 `true`。非静音的自动播放可能被浏览器策略拒绝。
- `opacity` — 叠加层的目标不透明度（淡入的终值）。默认 `1`。
- `playbackRate` — 初始播放速度。默认 `1`。
- `fit` — 视频的 CSS `object-fit`。默认 `"cover"`。
- `zIndex` — 多个 Vfx 叠加层之间的排序。默认 `0`。

参见 [Vfx](/zh/docs/narraleaf-react/core/elements/vfx)。
