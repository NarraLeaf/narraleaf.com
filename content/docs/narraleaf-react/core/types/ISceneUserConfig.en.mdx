---
title: "ISceneUserConfig"
---
```typescript
interface ISceneUserConfig {
    /**
     * Background music
     */
    backgroundMusic: Sound | null;
    /**
     * Background music fade duration, in milliseconds
     */
    backgroundMusicFade: number;
    /**
     * Voice map or a function that returns the voice URL
     */
    voices?: VoiceIdMap | VoiceSrcGenerator;
    /**
     * Background src, can be a {@link Color} or an {@link Image}
     */
    background: ImageSrc | Color;
    /**
     * An array of {@link Layer}s
     */
    layers: Layer[];
}
```

- For Sound, see [Sound](/docs/narraleaf-react/core/elements/sound)
- For VoiceIdMap and VoiceSrcGenerator, see [ISoundUserConfig](/docs/narraleaf-react/core/types/ISoundUserConfig).
- For ImageSrc, see [ImageSrc](/docs/narraleaf-react/core/types/ImageSrc).
- For Color, see [Color](/docs/narraleaf-react/core/types/color).
- For Layer, see [Layer](/docs/narraleaf-react/core/elements/layer).
