title: "SentenceUserConfig"

```typescript
type SentenceUserConfig = Partial<Omit<SentenceConfig, "voice"> & {
    voice: Sound | string | null | undefined;
}>;
```

参见 [SentenceConfig](/zh/docs/narraleaf-react/core/types/SentenceConfig)，其中包含 `pause`、`metadata`、`avatar`（对话框头像，见 [Dialog](/zh/docs/narraleaf-react/player/dialog)）等可选字段。
