---
title: "StorableData"
---
```typescript
type StorableData<K extends string = string> = {
    [key in K]: number | boolean | string | StorableData | StorableData[] | undefined | null | Date;
};
```

This is the shape of a namespace's contents in memory — what you read and write through [Namespace](/docs/narraleaf-react/core/game/storable/namespace).

It is not the shape a save carries. In a saved game each value is tagged with its type so that a `Date` survives JSON, see [SerializedNamespaceData](/docs/narraleaf-react/core/types/SerializedNamespaceData).
