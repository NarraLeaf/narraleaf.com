NarraLeaf-React 易于理解和使用。本指南将帮助您使用 NarraLeaf-React 创建您的第一个视觉小说。

## 创建你的第一个视觉小说

现在是时候使用 NarraLeaf-React 创建您的第一个视觉小说了。

<Steps>
  <Step>

### 创建一个新的 React 应用

首先，使用 Create React App 创建一个新的 React 应用。

导航到您想要创建应用的目录，并运行以下命令：

```bash
npx nlr-skeleton@latest my-project
```

这将使用 NarraLeaf-React 模板创建一个新的 React 应用。

  </Step>
  <Step>

### 运行这个应用

要预览应用程序，请使用以下命令运行应用程序：

<Tabs items={['npm', 'yarn', 'pnpm']}>
  <Tab value="npm">
    ```bash
    npm run dev
    ```
  </Tab>
  <Tab value="yarn">
    ```bash
    yarn dev
    ```
  </Tab>
  <Tab value="pnpm">
    ```bash
    pnpm dev
    ```
  </Tab>
</Tabs>

访问 `http://localhost:3000` 查看您的第一个视觉小说。

  </Step>
</Steps>

恭喜！您已经使用 NarraLeaf-React 创建了您的第一个视觉小说。

您可以编辑 `src/story.ts` 文件来创建您自己的故事！
