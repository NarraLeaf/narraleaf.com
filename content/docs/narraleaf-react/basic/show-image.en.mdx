---
title: "Show Image"
---
To show an image, you need to use the [Image](/docs/narraleaf-react/core/elements/image) element.

## Define an Image

You can construct an image by providing the [IImageUserConfig](/docs/narraleaf-react/core/types/IImageUserConfig) object.

```typescript
import {Image} from "narraleaf-react";
```

```typescript
const character1Happy = new Image({
    src: "https://YOUR_IMAGE_URL",
    zoom: 0.5, // The zoom of the image
    position: {
        yalign: 0.3, // The vertical alignment of the image
        xalign: 0.5, // The horizontal alignment of the image
    },
});
```

This image will be displayed at the left side of the screen with a scale of 0.5.

## Show the Image

You can show the image by adding actions to the scene.

```typescript
scene1.action([
    character1Happy.show(),
]);
```

The image will be displayed on the screen immediately.

You can show the image with ease. 

```typescript
scene1.action([
    character1Happy.show({
        ease: "easeInOut",
        duration: 1000,
    }),
]);
```

## Hide the Image

You can hide the image by adding actions to the scene.

```typescript
scene1.action([
    character1Happy.hide(),
]);
```

The image will be hidden from the screen immediately.

You can hide the image with ease. 

```typescript
scene1.action([
    character1Happy.hide({
        ease: "easeInOut",
        duration: 1000,
    }),
]);
```

## Change the Image

### Src-based Image

You can change the image by updating the image source.

```typescript
scene1.action([
    character1Happy.char("https://NEW_IMAGE_URL"),
]);
```

or with a [transition](/docs/narraleaf-react/core/animation/transition).

```typescript
import {Dissolve} from "narraleaf-react";
```

```typescript
scene1.action([
    character1Happy.char("https://NEW_IMAGE_URL", new Dissolve(300)),
]);
```

### Use Tag-based Image

You can use the tag-based image to show the image.

> Note: The tag-based image is a feature that allows you to change the image appearance by providing tags.
>
> You can't mix the tag-based image configuration with the normal image configuration.  
> Normal image configuration has a string as its `src` property.

```typescript
import {Image} from "narraleaf-react";
```

```typescript
const image = new Image({
    src: {
        groups: [
            ["happy", "sad", "angry"], // group 1
            ["shirt", "jacket", "t-shirt"], // group 2
            ["trousers", "skirt", "shorts"], // group 3
        ],
        defaults: ["happy", "shirt", "trousers"], // default appearance
        resolve: (emotion, top, bottom) => `https://your/image/src/${emotion}_${top}_${bottom}.png`,
    } as const,
});
// The default image will be `https://your/image/src/happy_shirt_trousers.png`

scene1.action([
    image.show(),
    image.char(["angry", "shorts"], new Dissolve(1000)), // change the image to `https://your/image/src/angry_shirt_shorts.png`
]);
```

This shape expects one pre-composited image per combination of tags. If your artwork is exported as one image per part instead, use a **layered image** — you write the parts once and NarraLeaf composes them, so 3 emotions × 3 tops × 3 bottoms is 9 files rather than 27.

```typescript
const image = new Image({
    src: {
        // bottom to top
        layers: [
            {trousers: "trousers.png", skirt: "skirt.png", shorts: "shorts.png"},
            {shirt: "shirt.png", jacket: "jacket.png", tshirt: "t-shirt.png"},
            {happy: "happy.png", sad: "sad.png", angry: "angry.png"},
        ],
        defaults: ["trousers", "shirt", "happy"],
    },
});

scene1.action([
    image.show(),
    image.char(["angry", "shorts"], new Dissolve(1000)), // only those two layers change
]);
```

Read more about tag-based and layered images, please read the [Image](/docs/narraleaf-react/core/elements/image) documentation.

## Animate the Image

You can animate the image with [Transform](/docs/narraleaf-react/core/animation/transform).

```typescript
import {Transform, CommonPosition} from "narraleaf-react";
```

```typescript
// Move the image to the right side of the screen
scene1.action([
    character1Happy.transform(
        Transform.create()
            .position("right")
            .commit({ duration: 120, ease: "easeOut" })
    ),
]);

// or

scene1.action([
    character1Happy.transform(Transform.left(120, "easeOut")),
]);
```

The image will move to the right side of the screen with a duration of 120 milliseconds and ease out effect.

For more information about the Transform, please check the [Transform](/docs/narraleaf-react/core/animation/transform) documentation.
