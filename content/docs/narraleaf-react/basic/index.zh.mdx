基础教程适合在你已经拥有一个可运行的 NarraLeaf-React 项目后阅读。它会带你完成故事编写的主要流程：定义场景、添加动作、显示对话和图片、连接故事段落，然后逐步加入交互、音频、数据、语音和玩家偏好设置。

## 核心流程

<Steps>
<Step>

### 创建场景

从定义 [Scene](./create-scene) 开始。场景会为视觉小说中的一段内容提供名称、背景和叙事上下文。

</Step>
<Step>

### 添加动作

使用[动作](./add-actions)描述场景中会发生什么。动作是 NarraLeaf-React 在故事播放时按顺序执行的指令。

</Step>
<Step>

### 呈现故事内容

显示[对话](./show-dialog)、展示[图片](./show-image)，并通过[播放故事](./play-story)把场景交给播放器运行。

</Step>
<Step>

### 加入交互和状态

使用[选项](./make-choices)、[条件](./conditional)和[存储数据](./store-data)，让故事能够响应玩家选择和之前发生过的事件。

</Step>
<Step>

### 打磨体验

加入[音效](./sound)、[语音](./voice)和[偏好设置](./manage-preferences)，让视觉小说体验更完整，并让玩家能够控制播放表现。

</Step>
</Steps>
