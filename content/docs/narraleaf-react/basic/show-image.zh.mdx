---
title: "显示图片"
---
要显示图片，你需要使用 [Image](/zh/docs/narraleaf-react/core/elements/image) 元素。

## 定义图片

你可以通过提供 [IImageUserConfig](/zh/docs/narraleaf-react/core/types/IImageUserConfig) 对象来构建图片。

```typescript
import {Image} from "narraleaf-react";
```

```typescript
const character1Happy = new Image({
    src: "https://你的图片链接",
    zoom: 0.5, // 图片的缩放比例
    position: {
        yalign: 0.3, // 图片的垂直对齐方式
        xalign: 0.5, // 图片的水平对齐方式
    },
});
```

这张图片将以 0.5 的比例显示在屏幕的左侧。

## 显示图片

你可以通过向场景添加动作来显示图片。

```typescript
scene1.action([
    character1Happy.show(),
]);
```

图片将立即显示在屏幕上。

你可以使用缓动效果显示图片。

```typescript
scene1.action([
    character1Happy.show({
        ease: "easeInOut",
        duration: 1000,
    }),
]);
```

## 隐藏图片

你可以通过向场景添加动作来隐藏图片。

```typescript
scene1.action([
    character1Happy.hide(),
]);
```

图片将立即从屏幕上隐藏。

你可以使用缓动效果隐藏图片。

```typescript
scene1.action([
    character1Happy.hide({
        ease: "easeInOut",
        duration: 1000,
    }),
]);
```

## 更改图片

### 基于 Src 的图片

你可以通过更新图片源来更改图片。

```typescript
scene1.action([
    character1Happy.char("https://新图片链接"),
]);
```

或者使用 [transition](/zh/docs/narraleaf-react/core/animation/transition)。

```typescript
import {Dissolve} from "narraleaf-react";
```

```typescript
scene1.action([
    character1Happy.char("https://新图片链接", new Dissolve(300)),
]);
```

### 使用基于标签的图片

你可以使用基于标签的图片来显示图片。

> 注意：基于标签的图片是一项功能，允许你通过提供标签来更改图片外观。
>
> 你不能将基于标签的图片配置与普通图片配置混合使用。
> 普通图片配置的 `src` 属性为字符串。

```typescript
import {Image} from "narraleaf-react";
```

```typescript
const image = new Image({
    src: {
        groups: [
            ["happy", "sad", "angry"], // 组 1
            ["shirt", "jacket", "t-shirt"], // 组 2
            ["trousers", "skirt", "shorts"], // 组 3
        ],
        defaults: ["happy", "shirt", "trousers"], // 默认外观
        resolve: (emotion, top, bottom) => `https://你的/图片/路径/${emotion}_${top}_${bottom}.png`,
    } as const,
});
// 默认图片将是 `https://你的/图片/路径/happy_shirt_trousers.png`

scene1.action([
    image.show(),
    image.char(["angry", "shorts"], new Dissolve(1000)), // 将图片更改为 `https://你的/图片/路径/angry_shirt_shorts.png`
]);
```

这种形式需要为每一种标签组合准备一张已合成好的图片。如果你的素材是按部件分别导出的，可以改用**层叠图像** —— 每个部件只需画一次，由 NarraLeaf 负责合成，于是 3 种表情 × 3 种上衣 × 3 种下装 只需 9 张图，而不是 27 张。

```typescript
const image = new Image({
    src: {
        // 从下到上
        layers: [
            {trousers: "trousers.png", skirt: "skirt.png", shorts: "shorts.png"},
            {shirt: "shirt.png", jacket: "jacket.png", tshirt: "t-shirt.png"},
            {happy: "happy.png", sad: "sad.png", angry: "angry.png"},
        ],
        defaults: ["trousers", "shirt", "happy"],
    },
});

scene1.action([
    image.show(),
    image.char(["angry", "shorts"], new Dissolve(1000)), // 只有这两个图层会改变
]);
```

有关基于标签的图片和层叠图像的更多信息，请阅读 [Image](/zh/docs/narraleaf-react/core/elements/image) 文档。

## 动画化图片

你可以使用 [Transform](/zh/docs/narraleaf-react/core/animation/transform) 对图片进行动画处理。

```typescript
import {Transform, CommonPosition} from "narraleaf-react";
```

```typescript
// 将图片移动到屏幕右侧
scene1.action([
    character1Happy.transform(
        Transform.create()
            .position("right")
            .commit({ duration: 120, ease: "easeOut" })
    ),
]);

// 或者

scene1.action([
    character1Happy.transform(Transform.left(120, "easeOut")),
]);
```

图片将以 120 毫秒的持续时间和缓出效果移动到屏幕右侧。

有关 Transform 的更多信息，请查看 [Transform](/zh/docs/narraleaf-react/core/animation/transform) 文档.
