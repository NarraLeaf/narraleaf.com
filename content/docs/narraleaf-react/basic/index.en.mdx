The Basic guide is the first place to go after you have a running NarraLeaf-React project. It walks through the story-writing workflow: define scenes, add actions, show dialog and images, connect scenes together, and then layer in interactivity, audio, data, voice, and player preferences.

## Core Workflow

<Steps>
<Step>

### Create a scene

Start by defining a [Scene](/docs/narraleaf-react/basic/create-scene). A scene gives a named section of your visual novel its own background and story context.

</Step>
<Step>

### Add actions

Use [actions](/docs/narraleaf-react/basic/add-actions) to describe what happens inside the scene. Actions are the ordered instructions NarraLeaf-React executes while the story is playing.

</Step>
<Step>

### Present story content

Show [dialog](/docs/narraleaf-react/basic/show-dialog), display [images](/docs/narraleaf-react/basic/show-image), and then [play the story](/docs/narraleaf-react/basic/play-story) through the player so the scene can be experienced interactively.

</Step>
<Step>

### Add interaction and state

Use [choices](/docs/narraleaf-react/basic/make-choices), [conditions](/docs/narraleaf-react/basic/conditional), and [stored data](/docs/narraleaf-react/basic/store-data) to make the story react to player decisions and previous events.

</Step>
<Step>

### Polish the experience

Add [sound](/docs/narraleaf-react/basic/sound), [voice](/docs/narraleaf-react/basic/voice), and [preferences](/docs/narraleaf-react/basic/manage-preferences) so the visual novel feels complete and gives players control over playback.

</Step>
</Steps>
