---
title: "添加动作"
---

场景按从上到下的顺序执行动作。一个场景只调用一次 `scene.action()`，把完整动作列表传进去。

```ts
import { Character, Scene } from "narraleaf-react";

const kitchen = new Scene("厨房", {
    background: "/backgrounds/kitchen.webp",
});
const street = new Scene("街道", {
    background: "/backgrounds/street.webp",
});
const alice = new Character("爱丽丝");

kitchen.action([
    alice.say("早餐准备好了。"),
    kitchen.jumpTo(street),
]);
```

大多数故事元素的方法都会返回动作，例如 `character.say()`、`image.show()`、`sound.play()` 和 `scene.jumpTo()`。动作列表也可以直接写字符串，字符串会作为旁白显示。

```ts
kitchen.action([
    "房间里很安静。",
    alice.say("走吧。"),
]);
```

需要多次引用当前场景时，可以使用回调写法：

```ts
import { Control } from "narraleaf-react";

const room = new Scene("房间").action((scene) => [
    scene.setBackground("/backgrounds/room-night.webp"),
    Control.sleep(500),
    scene.jumpTo(street),
]);
```

<Callout type="warning">
    不要对同一个场景多次调用 `action()`。`jumpTo()` 后面的动作不会执行。
</Callout>

完整的可用类型见 [ActionStatements](/zh/docs/narraleaf-react/core/types/ActionStatements)。
