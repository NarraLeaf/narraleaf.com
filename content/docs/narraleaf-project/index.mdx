NarraLeaf Project is the umbrella for three complementary solutions: **NarraLeaf Studio**, **NarraLeaf**, and **NarraLeaf-React**.

Use the product tabs to move between the ecosystem overview and each product's documentation. Engine APIs live under **NarraLeaf -> Library**: **Main**, **Renderer**, and **CLI**.
