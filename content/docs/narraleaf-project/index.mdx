NarraLeaf Project is the umbrella for three complementary solutions: **NarraLeaf Studio**, **NarraLeaf** (the engine and tooling — open the **NarraLeaf** documentation tab), and **NarraLeaf-React**.

Use the **documentation tabs** (sidebar switcher) to move between this overview and each product’s docs. Engine APIs live under **NarraLeaf → Library** (**Main**, **Renderer**, **CLI**).
