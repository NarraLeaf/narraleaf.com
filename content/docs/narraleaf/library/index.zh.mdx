NarraLeaf 是 NarraLeaf Project 生态中的桌面应用路径。它将运行时、渲染层和命令行工具组合在一起，用于构建和交付完整的 NarraLeaf 应用。

当你正在构建完整的视觉小说应用，并且需要创作体验之下的引擎级能力时，请使用本文档。

## 何时选择这一路径

如果你的项目需要完整的桌面应用工具链，包括运行时行为、渲染、项目命令和交付流程，请选择 NarraLeaf。

如果你的首要目标是通过引导式可视化环境进行零代码故事创作和制作管理，请选择 [NarraLeaf Studio](/zh/docs/studio)。

如果你已经有 React 应用，需要轻量嵌入式播放器，或希望完全掌控周围的 Web UI，请选择 [NarraLeaf-React](/zh/docs/narraleaf-react)。

## 推荐阅读顺序

先阅读 [Main](./main)，理解运行时和创作模型。然后阅读 [Renderer](./renderer)，了解场景和界面如何呈现。最后阅读 [CLI](./cli)，在准备创建、运行和交付 NarraLeaf 应用时使用。

## 生态

NarraLeaf 是更大的 [NarraLeaf Project](/zh/project) 的一部分。你也可以查看 [NarraLeaf Studio](/zh/docs/studio) 了解可视化创作路径，或查看 [NarraLeaf-React](/zh/docs/narraleaf-react) 了解基于 React 的嵌入方案。
