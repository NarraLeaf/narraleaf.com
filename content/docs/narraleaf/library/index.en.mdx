NarraLeaf is the desktop application path in the NarraLeaf Project ecosystem. It brings together the runtime, rendering layer, and command-line tooling used to build and deliver complete NarraLeaf applications.

Use this documentation when you are building a full visual novel application and need the engine-level pieces that sit below the authoring experience.

## When to Use This Path

Choose NarraLeaf when your project needs a complete desktop application toolchain, including runtime behavior, rendering, project commands, and delivery workflow.

Choose [NarraLeaf Studio](/docs/studio) when your first goal is zero-code story authoring and production through a guided visual environment.

Choose [NarraLeaf-React](/docs/narraleaf-react) when you already have a React application, need a lightweight embedded player, or want to own the surrounding web UI.

## Recommended Reading Path

Start with [Main](/docs/narraleaf/library/main) to understand the runtime and authoring model. Continue with [Renderer](/docs/narraleaf/library/renderer) to see how scenes and interfaces are presented. Finish with [CLI](/docs/narraleaf/library/cli) when you are ready to scaffold, run, and ship a NarraLeaf application.

## Ecosystem

NarraLeaf is one part of the broader [NarraLeaf Project](/project). You can also explore [NarraLeaf Studio](/docs/studio) for visual authoring and [NarraLeaf-React](/docs/narraleaf-react) for React-based embedding.
