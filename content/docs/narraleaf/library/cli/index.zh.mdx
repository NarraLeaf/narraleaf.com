**CLI** 包提供 NarraLeaf 桌面项目使用的 `narraleaf` 命令。

```bash
narraleaf init my-visual-novel
cd my-visual-novel
narraleaf dev
```

## CLI 会读取什么

每个项目命令都会从当前工作目录运行，并期望存在：

- `package.json`
- `narraleaf.config.js`
- 配置的主进程入口
- 配置的渲染器目录
- `renderer/App.tsx` 或 `renderer/App.jsx`
- `renderer/pages`
- `renderer/public`

完整配置结构参见 [项目配置](/zh/docs/narraleaf/library/project-config)。
