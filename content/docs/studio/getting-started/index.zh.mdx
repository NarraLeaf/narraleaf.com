本指南从启动器创建一个空白项目。完成后，Studio 会打开项目工作区，并为 UI 编辑器准备一个 `Main Page`。

## 创建项目

<Steps>
<Step>

### 打开项目向导

在启动器的**项目**页选择**新建项目**。

向导分为模板、详情、设置和确认四步。当前可以从**空白**模板开始。
</Step>
<Step>

### 填写项目详情

至少填写以下内容：

- **项目名称**：Studio 和构建产物中显示的项目名。
- **应用 ID**：只使用小写字母、数字和连字符。创建后不宜更改。
- **舞台尺寸**：游戏 UI 和运行画面的基准尺寸。

作者、许可证和项目描述可以稍后补充。
</Step>
<Step>

### 选择项目位置

选择一个空目录，或填写一个尚不存在的目录。Studio 不会在非空目录中创建项目。

如需使用 Git，在此处启用版本控制。版本控制用于记录项目文件的变化，不能代替素材源文件的外部备份。
</Step>
<Step>

### 确认并创建

在**确认**页检查项目名称、应用 ID、舞台尺寸和保存位置，然后选择**创建项目**。

创建完成后，Studio 会直接打开工作区。
</Step>
</Steps>

## 首次打开

空白项目并非完全没有内容。UI 编辑器会创建并打开 `Main Page`，它是项目的默认主页面。可以直接在这个页面上制作标题界面，也可以另建页面。

项目的主要入口位于左侧活动栏：

- **项目**：元数据、图标、依赖和项目设置。
- **故事**：故事、章节和场景。
- **UI**：页面、游戏 UI 和组件库。
- **角色**：角色、形态和差分。
- **素材**：项目素材及分组。
- **本地化**：语言、翻译和审校。

## 打开已有项目

在启动器中选择**打开项目**或**打开文件夹**，然后选择包含 `.nlproj` 文件的项目目录。若收到 `.nlspkg` 项目包，使用**导入项目**并选择一个空目录解包。
