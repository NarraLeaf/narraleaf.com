title: 认识工作区

---

工作区由活动栏、侧边栏、编辑器、属性面板和底部面板组成。选择左侧活动栏中的功能会切换侧边栏；打开的页面、故事场景和蓝图会出现在中央编辑区。

## 主要区域

| 区域 | 作用 |
| --- | --- |
| 左侧活动栏 | 在项目、故事、UI、角色、素材和本地化之间切换 |
| 左侧边栏 | 浏览和创建当前功能中的内容 |
| 中央编辑区 | 以标签页打开页面、场景、素材预览和蓝图 |
| 右侧属性面板 | 编辑当前所选页面、控件、素材或故事动作的属性 |
| 底部面板 | 显示控制台、构建输出、动画和可选的素材面板 |
| 顶部操作栏 | 打开文件操作、开发模式、预览和构建 |

可以从**窗口**菜单分别显示或隐藏左右侧边栏和底部面板。

## 选择与属性

属性面板始终跟随当前选择。比如：

- 在 UI 画布中选择文本，右侧显示文字、布局和外观属性。
- 在素材面板中选择图片，右侧显示素材信息。
- 在故事场景中选择动作，右侧显示该动作的参数。

如果右侧显示“未选择任何项目”，先在当前编辑器或侧边栏中选中要编辑的对象。

## 编辑器标签页

双击或打开内容后，Studio 会在中央区域创建标签页。标签页可关闭、重新打开，也可以通过 `Ctrl+Tab` 在最近使用的标签页之间切换。关闭工作区后，Studio 会恢复支持持久化的标签页。

标签页标题旁的修改标记表示内容尚未写入项目文件。Studio 会自动保存多数编辑，但在启动开发模式或构建前，仍应留意错误通知和静态检查提示。

## 调整工作区

- 拖动区域边缘可调整侧边栏和底部面板尺寸。
- 使用**窗口 > 显示侧边栏**、**显示右边栏**或**显示底边栏**恢复被隐藏的区域。
- 关闭一个面板不会删除其中的项目内容。
- 控制台默认位于底部面板；开发模式和构建的详细输出会显示在这里。

<Callout type="info">
画布预览主要用于排版。蓝图执行、节点跟踪和实际游戏状态以开发模式中的结果为准。
</Callout>
