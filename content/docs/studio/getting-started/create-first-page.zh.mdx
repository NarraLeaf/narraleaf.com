---
title: 创建第一个页面
description: 使用 Main Page 制作一个可运行的简单标题界面。
---

新建项目会自动创建 `Main Page`。本页直接编辑这个默认页面，放置背景、标题和按钮，并在开发模式中检查结果。

## 打开 Main Page

<Steps>
<Step>

### 打开 UI 面板

在左侧活动栏选择 **UI**，确认界面类型为**页面**，然后打开 `Main Page`。

页面会在中央编辑区中打开。左侧是 UI 大纲，中央是画布，右侧是属性面板，画布底部是插入栏。
</Step>
<Step>

### 添加背景容器

在底部插入栏选择 **Container**，再在画布上单击或拖出一个区域。选中容器后，在右侧属性面板中调整位置、尺寸和背景。

若背景使用图片，也可以把图片素材直接拖到画布。Studio 会创建一个 Image 控件并关联该素材。
</Step>
<Step>

### 添加标题

从插入栏选择 **Text**，在画布上放置文本。双击文本可直接编辑内容；字体、字号、颜色和对齐方式在右侧属性面板中设置。

拖动控件可移动位置，拖动边框控制点可改变尺寸。启用智能吸附后，控件会对齐画布边缘、中心线和相邻控件。
</Step>
<Step>

### 添加按钮

选择 **Button** 并放置到画布。先完成按钮的文字和外观；按钮行为可以随后在控件蓝图中连接。

若只是检查布局，按钮暂时不需要绑定事件。
</Step>
</Steps>

## 使用 UI 大纲

UI 大纲显示页面中的控件层级。容器可以包含其他控件；子控件会随父容器一起移动。大纲也适合处理画布上重叠或尺寸很小、难以直接点选的控件。

右键控件可以重命名、显示或隐藏、调整前后顺序，也可以把选中内容加入组件库。准备复用一组控件时，再将其制作成组件。

## 撤销与保存

- 使用 `Ctrl+Z` 撤销，使用 `Ctrl+Shift+Z` 重做。
- `Ctrl+C`、`Ctrl+V` 可复制和粘贴控件。
- 删除页面会同时移除其中的全部控件；`Main Page` 不能删除。
- 页面数据会写入项目目录中的 UI 文档，不需要另存为单独文件。

## 检查页面

选择画布右上角的播放按钮，可在开发模式中打开当前页面。开发模式使用实际运行环境，适合检查比例、交互和蓝图行为。

下一步参阅[运行与打包](/zh/docs/studio/getting-started/run-and-package)。
